# 📊 State-wise Elementary Education Analysis (India)

## 📌 Overview

This project analyzes elementary education data across Indian states for the academic year **2015–16**. The goal is to uncover patterns in **enrollment, dropout rates, and regional disparities** using data analytics techniques.

---

## 🎯 Problem Statement

Education outcomes vary significantly across Indian states. High enrollment does not always translate into sustained education, as dropout rates remain a critical issue.

This project aims to:

* Identify patterns in enrollment and dropout rates
* Compare state-wise performance
* Highlight disparities in education outcomes

---

## 📂 Dataset

* Source: Government education dataset (2015–16)
* Level: State-wise elementary education
* Format: CSV

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 🔍 Project Workflow

### 1. Data Cleaning

* Handled missing values
* Checked data consistency
* Prepared dataset for analysis

### 2. Exploratory Data Analysis (EDA)

* Analyzed enrollment trends
* Compared states across key metrics
* Visualized patterns using charts and graphs

### 3. Data Visualization

* Bar charts for state comparison
* Heatmaps for correlation analysis
* Distribution plots for key variables

---

## 📈 Key Insights

* Some states show **high enrollment but also high dropout rates**, indicating weak retention systems.
* Significant **regional disparities** exist in education performance across India.
* Certain states consistently perform better, suggesting **stronger policy implementation and infrastructure**.
